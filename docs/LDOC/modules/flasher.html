Flasher:eraseArea (tPlugin, aAttr, ulDeviceOffset, ulSize, fnCallbackMessage, fnCallbackProgress)
erase an area: check if the area is already erased and erase only if it isn't empty. ulSize = 0xffffffff to erase from ulDeviceOffset to end of chip

OK: The area is empty, no erase necessary. Area erased

Error messages: getEraseArea failed! Failed to check if the area is erased! getEraseArea failed! Failed to erase the area! (Failure during erase) Failed to erase the area! (isErased check failed)

Flasher:hashArea (tPlugin, aAttr, ulDeviceOffset, ulDataByteSize, fnCallbackMessage, fnCallbackProgress)
Calculate the SHA1 hash of an area of an area in the flash. size = 0xffffffff to read from ulDeviceOffset to end of device

Returns the hash as a binary string or nil and an error message.

Ok: "Checksum calculated."

Error messages: Could not determine the flash size! "Error while calculating SHA1 hash"
